File manager panel

Browses the filesystem and distinguishes tracked files (those with a metarecord) from untracked ones, so you can bring files under management.

Navigating

  • ↑ up / repo root — move through directories.
  • Arrow keys (or j/k) move the cursor; double-click or Enter opens a directory.
  • ⟳ refresh re-reads the current directory.
  • constrain to repo root keeps browsing inside the active repository.

Tracking files

Tracking is opt-in: nothing is watched until mf_watch is enabled for a directory (inherited down the tree, filtered by mf_ignore). Use Add to DB to create metarecords for the selected files. Once tracked, files appear in the metarecord list, and the daemon keeps their mfr_path in sync as they move. To bulk-import existing files, run reconcile:run.
