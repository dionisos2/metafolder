Getting started

metafolder organises your files as metarecords — records with named fields — kept in sync with a directory tree. This GUI is a set of panels arranged in workspaces, driven by a command input and keyboard shortcuts.

Using help

  • Run help (or help:help) to open this panel.
  • help:help "topic" jumps straight to a page when the text is an exact name (a panel type, a command, an alias); otherwise it searches.
  • Prefix the search box with # to force a text search.
  • help:help-cursor turns the pointer into a ?; click any part of the window to open the matching help page, or press Escape to cancel.

The interface

  • Tab bar — one tab per workspace; the tab:* and workspace:* commands create and move between them.
  • Panel slots — one or two side-by-side panels. Pick a panel type from the header selector, or use panel:set-type, panel:split and panel:focus-next. Drag the divider to resize.
  • Command input — always visible; type a command (with autocomplete) or ! to run a shell command (with % placeholders for the selection).
  • Status bar — per-workspace messages and the active key hints; the task bar shows background progress.
  • Settings — config:open shows config paths and the keybinding editor.

Panels

Each panel type has its own help page:

  • Repositories — load and manage repositories.
  • Metarecord list — query and list metarecords.
  • Metarecord detail — view and edit one metarecord.
  • File manager — browse the filesystem and track files.
  • File — preview a file and its fields.
  • Tree reference — navigate a TreeRef forest.
  • Reference list — metarecords referencing a node.
  • Log — the event log; undo/redo.
  • Message, Workspace info, Hello example.
